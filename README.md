## Step One
### Installing NPM Dependencies!

``Option one(Simple)``
1. Run ``SETUP.bat``(WINDOWS) Or ``STEUP.sh``(LINUX) 
2. Wait For Install

``Option Two(Harder)``
1. Open CMD/Terminal
2. Enter the following command:
```
npm i discord.js glob
```
3. Wait For Install
4. After Installing run command:
```
node .
``` 
With Sharding Manager

OR
```
npm run noshard
```
Without Sharding Manager
